# Contributing

SecApp is at an early design stage. Keep contributions focused, reviewable, and consistent with the security and consent boundaries described in the repository.

Before contributing:

- do not add Velociraptor binaries or a modified Velociraptor fork;
- do not commit real audit results, evidence, personal or system data, credentials, tokens, or other secrets;
- do not use real sensitive data as a test fixture;
- keep collection, risk assessment, and user-interface concerns separate;
- preserve read-only behavior as the default and require explicit consent for sensitive operations;
- discuss substantial design or scope changes before implementation.

Contributions are submitted under the repository license, `AGPL-3.0-or-later`.
